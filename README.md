# GlobalThink Challenge


## Project setup
```
yarn install
```

## Create .env files
```
cp .env.example .env
cp .env.test.example .env.test
```

## Compile and run the project

```bash
# development
yarn run start

# dev mode
yarn run start:dev
```

## Run tests

```bash
# unit tests
yarn run test

# e2e tests
yarn run test:e2e
```

## Run with docker
```
docker compose up -d
```

### Backend running on
[http://localhost:3000/api](http://localhost:3000/api)
